# Contributing

Contributions are welcome!

Before submitting a pull request, please make sure that:

- The app is available on macOS.
- The entry follows the existing format: `- [Name](link) - Description.`
- The description is a short, factual sentence ending with a period.
- The entry is placed in the most relevant category, in alphabetical order.
- The link points to the official website or App Store page, using `https://` when available.
- New categories are only added when no existing one fits.

If in doubt, open an issue to discuss before submitting a pull request.
